# 📁 Dados — Instruções

Os CSVs do dataset **não estão versionados** neste repositório por questões de tamanho (>80MB).

## Como baixar

1. Acesse: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
2. Clique em **Download** (requer conta gratuita no Kaggle)
3. Extraia o arquivo `.zip`
4. Coloque os 9 CSVs na pasta `data/raw/`

## Arquivos esperados

```
data/raw/
├── olist_customers_dataset.csv
├── olist_geolocation_dataset.csv
├── olist_order_items_dataset.csv
├── olist_order_payments_dataset.csv
├── olist_order_reviews_dataset.csv
├── olist_orders_dataset.csv
├── olist_products_dataset.csv
├── olist_sellers_dataset.csv
└── product_category_name_translation.csv
```

## Fonte

- **Dataset:** Brazilian E-Commerce Public Dataset by Olist
- **Licença:** CC BY-NC-SA 4.0
- **Período:** Setembro 2016 – Outubro 2018
- **Volume:** ~100 mil pedidos reais anonimizados
